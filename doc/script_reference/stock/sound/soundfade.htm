SoundFade( <volume scale>, <time> )

Module: Sound

Summary:

Scales the sound volume by the given factor, for all players and channels except voice chat. If present, the scaling is linearly executed over the time provided via the second parameter, or instantly by default.



Example:

SoundFade( 3 );



Required Args

  • 1 : <volume scale> The factor (float) by which the sound volume should be affected.


Optional Args

  • 1 : <time> The time in seconds (float) over which the sound will fade/scale in seconds. If this option is not present then the music will stop/scale immediately.
