SetCvar( <cvar>, <value>, <serverinfo> )

Module: Dvars

Summary:

Sets the value of a script cvar. Can not set code cvars. Script cvars are not preserved in savegames.



Example:

SetCvar( "r_eyesAdjust", "1" );



Required Args:

  • 1 : <cvar> The cvar name as a string.
  • 2 : <value> The cvar value.


Optional Args

  • 1 : <serverinfo> If true, this cvar is published via getstatus responses. Default: False
