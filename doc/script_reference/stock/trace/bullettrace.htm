BulletTrace( <start>, <end>, <hit characters>, <ignore entity> )

Module: Trace

Summary:

Allows script to do a point trace with MASK_SHOT. Returns distance fraction, hit position, hit entity, hit surface normal vector, and hit surface type.



Example:

trace = BulletTrace( magicBulletOrigin.origin, eyePos, true, undefined );
/*
trace["fraction"] // float between including 0.0 and 1.0, stating how much of the trace distance was passed
trace["position"] // end position, if no hit
trace["entity"] // undefined, if no hit
trace["normal"] // normalized bullet direction, if no hit
trace["surfacetype"] // "none", if no hit
*/



Required Args:

  • 1 : <start> The bullet start point
  • 2 : <end> The bullet end point
  • 3 : <hit characters> When set to true, this will trace for character hits
  • 4 : <ignore entity> An entity to ignore
