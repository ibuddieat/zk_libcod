Debug Prints

Module: Callbacks

Summary:

Calls CodeCallback_DPrintf() in maps/mp/gametypes/_callbacksetup.gsc on debug messages (calls to Com_DPrintf), even if the developer dvar has value zero. Refrain from using this callback together with an excessive amount of debug messages since the callback allocates (and potentially exceeds the limit of) internal script variables. Only called for debug prints occurring within the main server thread, therefore debug prints from the proxy and asynchronous MySQL threads to not pass into this script callback function.

Note: The existance of the callback function is checked only once on server start. Functions added later are not taken into account, even after a map switch.



Parameters:

  • 1 : <message> Debug message to print.


Example:

CodeCallback_DPrintf(message)
{
    if(isSubStr(message, "rejected on a too high ping"))
        logPrintConsole(message);
}
