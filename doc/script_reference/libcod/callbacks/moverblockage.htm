Mover Blockage

Module: Callbacks

Summary:

Calls CodeCallback_MoverBlockage() in maps/mp/gametypes/_callbacksetup.gsc on entities that block motion of script mover entities (e.g., elevators). The first parameter is the entity that is being blocked (type script_brushmodel, script_model, or script_origin).

Note: The existance of the callback function is checked only once on server start. Functions added later are not taken into account, even after a map switch.



Called on:

The blocking entity



Parameters:

  • 1 : <mover> The mover entity that is being blocked.


Example:

CodeCallback_MoverBlockage(mover)
{
    if(isPlayer(self))
        self suicide();
}
