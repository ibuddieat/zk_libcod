Suicide

Module: Callbacks

Summary:

Calls CodeCallback_Suicide() in maps/mp/gametypes/_callbacksetup.gsc when suicide is called on a player. Call processSuicide on the player to proceed with the suicide operation.

Note: The existance of the callback function is checked only once on server start. Functions added later are not taken into account, even after a map switch.



Called on:

The player



Example:

CodeCallback_Suicide()
{
    // Room for further actions before processing the suicide
    self processSuicide();
}
