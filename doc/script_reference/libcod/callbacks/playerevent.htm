Player Event

Module: Callbacks

Summary:

Calls CodeCallback_PlayerEvent() in maps/mp/gametypes/_callbacksetup.gsc for each predictable player event (e.g., weapon firing, footstep sounds).

Note: The existance of the callback function is checked only once on server start. Functions added later are not taken into account, even after a map switch.



Called on:

The player



Parameters:

  • 1 : <event> The player event (integer). A list of possible events can be found at entity_event_t in: [link]
  • 2 : <parameter> An optional event parameter (integer)


Example:

CodeCallback_PlayerEvent(event, parameter)
{
    iprintLn("event " + event + " with parameter " + parameter + " for player " + self GetEntityNumber());
}
