Weapon Change

Module: Callbacks

Summary:

Calls CodeCallback_WeaponChange() in maps/mp/gametypes/_callbacksetup.gsc when the player switches between weapons (including to "none"). Not called on player spawn. Not called when the player drops weapons.

Note: The existance of the callback function is checked only once on server start. Functions added later are not taken into account, even after a map switch.



Called on:

The player



Example:

CodeCallback_WeaponChange(index)
{
    iprintLn(self.name + " switched to weapon with index " + index);
}
