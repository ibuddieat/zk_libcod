execute_async_create( <command>, <func> )

Module: Exec

Summary:

Executes a shell command in another thread and returns the result of it. Requires to use execute_async_checkdone once every server frame to poll the result.



Example:

execute_async_create("ls", ::resultCallback);



Required Args:

  • 1 : <command> Command to execute.
  • 2 : <func> Callback function with result array argument
