sightTracePassed( <start>, <end>, <hit characters>, <ignore entity>, <content mask> )

Module: Trace

Summary:

Allows script to do a point trace with MASK_OPAQUE_AI (by default). Returns true if trace complete, else returns false. Allows to use a custom content mask.



Example:

success = sightTracePassed( magicBulletOrigin.origin, eyePos, true, undefined, 41953329 /* 0x2802831 */ );



Required Args:

  • 1 : <start> The sight start point
  • 2 : <end> The sight end point
  • 3 : <hit characters> When set to true, this will trace for character hits
  • 4 : <ignore entity> An entity to ignore


Optional Args

  • 1 : <content mask> A custom content mask to use instead of MASK_OPAQUE_AI
