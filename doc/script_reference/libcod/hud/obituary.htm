obituary( <victim>, <attacker>, <weapon>, <means of death>, <team>, <origin>, <max. distance> )

Module: HUD

Summary:

Create an obituary for a character. Returns the (temporary) obituary entity. The optional parameters allow the obituary hud to be shown for a specific set of teams only, and/or to players that are in a certain range of a defined position only. Valid teams are:

  • "allies", "axis", "spectator"
  • combinations of two teams like: "allies+axis" or "spectator+axis"



Example:

obituary(self, attacker, sWeapon, sMeansOfDeath, "allies", self.origin, 512);



Required Args:

  • 1 : <victim> The victim entity
  • 2 : <attacker> The attacker entity
  • 3 : <weapon> The weapon name
  • 4 : <means of death> The means of death as a string


Optional Args

  • 1 : <team> The team to show the hud to
  • 2 : <origin> The position to use for max. distance comparison (requires the 3rd optional parameter)
  • 3 : <max. distance> The maximum 3D distance players have to be located within to be able to see the hud
