notSolidForPlayer( <player> )

Module: Entity

Summary:

Unsets the solid flag so that this object is no longer collidable, for the specified player. Requires the brush model collision tweak system to be enabled using the g_brushModelCollisionTweaks dvar. Resets on map load or a fast_restart.



Call this on:

A brush model entity



Example:

platform = getEnt("platform", "targetname");
platform notSolidForPlayer(player);



Required Args:

  • 1 : <player> The player for which the object should no longer be solid for
