enableBounce( <parallelBounce>, <perpendicularBounce> )

Module: Entity

Summary:

Enables gravity bouncing like grenades. Requires the use of enableGravity prior to the use of this method. Returns true on success, else false (gravity not enabled) or undefined if the entity is not a script_model.



Call this on:

script_model



Example:

// If self is a player:
e = spawn("script_model", self.origin + (0, 0, 256));
e setModel("xmodel/prop_bear_detail_sitting"); // Needs to be precached
e enableGravity();
e enableBounce();



Optional Args

  • 1 : <parallelBounce> Parallel bounce factor (float). Default: 0.5
  • 2 : <perpendicularBounce> Perpendicular bounce factor (float). Default: 0.25
