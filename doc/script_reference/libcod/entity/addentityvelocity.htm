addEntityVelocity( <velocity> )

Module: Entity

Summary:

Add velocity to the entity's velocity. Requires the use of enableGravity prior to the use of this method. Returns true on success, false if gravity is not enabled and undefined if the entity is not a script_model.



Call this on:

script_model



Example:

e addEntityVelocity((0, 0, 100));



Required Args:

  • 1 : <velocity> Velocity (vector) to add to the entity's velocity
