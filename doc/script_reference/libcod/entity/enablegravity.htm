enableGravity( <collide models>, <with rotation> )

Module: Entity

Summary:

Enables gravity similar to movegravity, but continuously and with collision. The initial maximum entity velocity is 8192 units per second (see setMaxEntityVelocity). A script notify with string 'land' is emit for the entity when landing, with the surface type as first and the ground entity as second parameter. Returns true on success or undefined if the entity is not a script_model. The following script methods will disable gravity (see disableGravity) automatically if called on the entity:

  • LinkTo (unless used as parent)
  • MoveGravity
  • MoveTo
  • MoveX
  • MoveY
  • MoveZ
  • RotatePitch
  • RotateRoll
  • RotateTo
  • RotateVelocity
  • RotateYaw



Call this on:

script_model



Example:

// If self is a player:
e = spawn("script_model", self.origin + (0, 0, 256));
e setModel("xmodel/prop_bear_detail_sitting"); // Needs to be precached
e enableGravity();



Optional Args

  • 1 : <collide models> Flag stating whether to also collide with other script_models. Default: True
  • 2 : <with rotation> Flag stating whether to also apply rotation to the target entity's angles. Default: True
