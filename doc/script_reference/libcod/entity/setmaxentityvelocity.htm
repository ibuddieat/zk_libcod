setMaxEntityVelocity( <velocity> )

Module: Entity

Summary:

Set the entity's maximum velocity in units per second. Requires the use of enableGravity prior to the use of this method. Returns true on success, false if gravity is not enabled and undefined if the entity is not a script_model.



Call this on:

script_model



Example:

e setMaxEntityVelocity(200);



Required Args:

  • 1 : <velocity> The entity's maximum velocity (float) in units per second
