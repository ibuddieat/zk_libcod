roundTo( <value>, <precision> )

Module: Math

Summary:

Returns the nearest integer value to the given value (in floating-point format), rounding halfway cases away from zero, and with the given precision.



Example:

num = roundTo(3.1415, 2.0);



Required Args:

  • 1 : <value> The value to round
  • 2 : <precision> The precision to apply
