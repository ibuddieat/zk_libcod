canUseTurret( <turret> )

Module: Turret

Summary:

Returns whether the player can use the specified turret. This would not be the case if the turret is already active, or if the player is not located behind the turret, or if the player is holding a grenade, or if the player is not on the ground.



Call this on:

A player



Example:

ready = self canUseTurret(turret);



Required Args:

  • 1 : <turret> The turret entity to probe for usage
