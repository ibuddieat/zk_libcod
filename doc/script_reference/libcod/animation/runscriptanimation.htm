runScriptAnimation( <event id>, <isContinue>, <force> )

Module: Animation

Summary:

Runs an animation on the player, based on the current player state. Returns the animation time or -1 if invalid.



Call this on:

The player



Example:

deathAnimDuration = self runScriptAnimation(1, 0, 1);
self [[level.callbackPlayerKilled]](eInflictor, eAttacker, iDamage, sMeansOfDeath, sWeapon, vDir, sHitLoc, psOffsetTime, deathAnimDuration);



Required Args:

  • 1 : <event id> The event id to use for the animation (e.g., 1 for death animations)
  • 2 : <isContinue> The isContinue flag
  • 3 : <force> The force flag
