setOriginAndAngles( <origin>, <angles> )

Module: Player

Summary:

Sets the player's origin and angles to the specified values, in a smooth way without delay, as opposed to SetOrigin.



Call this on:

The player



Example:

player setOriginAndAngles(self.origin + (0, 0, 64), (0, 90, 0));



Required Args:

  • 1 : <origin> The player's new origin (a point)
  • 2 : <angles> The player's new angles in degrees
