setBulletModel( <model>, <time> )

Module: Player

Summary:

Enables visual representation of dropping bullets (see enableBulletDrop) by attaching a model to each bullet. Requires the model to be precached. The optional time parameter defines how long that model remains visible after hitting something (default: 1000 milliseconds). Returns true on success, else undefined.



Call this on:

The player



Example:

player setBulletModel("xmodel/weapon_temp_panzershreck_rocket", 1000); // Needs to be precached



Required Args:

  • 1 : <model> Path to precached xmodel


Optional Args

  • 1 : <time> Time in milliseconds the model remains visible after hitting something
