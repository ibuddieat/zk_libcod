setConfigStringForPlayer( <index>, <string> )

Module: Player

Summary:

Sets a configstring, but only for the specified player.



Call this on:

The player



Example:

player setConfigStringForPlayer(index, "New Text");



Required Args:

  • 1 : <index> Index of the precached string that should be replaced/updated (see /configstrings in client console)
  • 2 : <string> The new value to replace the precached string with
