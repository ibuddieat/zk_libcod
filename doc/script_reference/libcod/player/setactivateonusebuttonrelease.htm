setActivateOnUseButtonRelease( <enabled> )

Module: Player

Summary:

Enables/disables whether the activation of entities (e.g., triggers, turrets, dropped weapons) is done on release of the use button, or otherwise already when it is pressed.



Call this on:

The player



Example:

player setActivateOnUseButtonRelease(true);



Required Args:

  • 1 : <enabled> Flag stating whether the activation of entities is done on release or press of the use button
