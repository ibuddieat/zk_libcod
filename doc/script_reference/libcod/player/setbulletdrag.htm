setBulletDrag( <value> )

Module: Player

Summary:

Sets the player's bullet slowdown (see enableBulletDrop). The default value is 0.01 (one percent per server frame). Returns the previous drag value.



Call this on:

The player



Example:

player setBulletDrag(0.01);



Required Args:

  • 1 : <value> Number of type float defining the bullet slowdown in percent per server frame
