executeClientCommand( <command> )

Module: Player

Summary:

Executes a text command on the client like via console input. However, there is some side effects for the player:

  • All sounds are reset
  • All effects are reset (disappear)
  • Aim-Down-Sight is reset



Called on:

The player



Parameters:

  • 1 : <command> Text command to execute on the client


Example:

self executeClientCommand("screenshotJPEG");
