setBulletVelocity( <value> )

Module: Player

Summary:

Sets the player's initial bullet velocity (see enableBulletDrop). The default value is 31500 units per second (about 800 meters per second). Returns the previous velocity value.



Call this on:

The player



Example:

player setBulletVelocity(31500);



Required Args:

  • 1 : <value> Number of type float defining the initial bullet velocity in units per second
