setStepSize( <size> )

Module: Player

Summary:

Sets the player's max. step size during stand and crouch stance (default: 18.0). Does not affect jump height. Revert this by setting the first argument to undefined. Returns true on success, or undefined on error.



Call this on:

A player



Example:

player setStepSize(36.0); // Double of default



Required Args:

  • 1 : <size> A float value defining the player's step size
