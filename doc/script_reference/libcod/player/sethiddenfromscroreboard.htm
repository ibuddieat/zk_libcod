setHiddenFromScoreboard( <enabled> )

Module: Player

Summary:

Enables/disables whether the player should be listed in the scoreboard and in server status responses. If the hidden player is the only player on the server, team scores in the scoreboard are invisible too.



Call this on:

The player



Example:

player setHiddenFromScoreboard(true);



Required Args:

  • 1 : <enabled> Flag stating whether the player should be listed in the scoreboard and in server status responses
