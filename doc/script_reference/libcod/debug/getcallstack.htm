getCallStack()

Module: Debug

Summary:

Returns the current call stack, which is an array consisting of filename and line number pairs. The first two elements describe the call to getCallStack() itself.



Example:

stack = getCallStack();
logPrintConsole("Call stack:\n");
for(i = 0; i < stack.size - 1; i += 2)
    logPrintConsole("> Line " + stack[i + 1] + " in " + stack[i] + "\n");
