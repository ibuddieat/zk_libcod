setConfigString( <index>, <string> )

Module: System

Summary:

Sets a configstring using the specified index, no matter if the configstring was empty or not. The index ranges for the respecive resources are allocated as follows:

  • Status icons: 23 - 30
  • Head icons: 31 - 45
  • Tags (populated via Attach): 110 - 141
  • Models: 334 - 589
  • Soundaliases (populated via PlaySound): 590 - 845
  • Effects: 846 - 909
  • Entity effects (populated via PlayFxOnTag): 910 - 1165
  • Shellshocks: 1166 - 1181
  • Menus: 1246 - 1277
  • Hint strings: 1278 - 1309
  • Strings: 1310 - 1565
  • Shaders: 1566 - 1693



Example:

player setConfigString(index, "New Text");



Required Args:

  • 1 : <index> Index of the precached string that should be replaced/updated (see /configstrings in client console)
  • 2 : <string> The new value to replace the precached string with
