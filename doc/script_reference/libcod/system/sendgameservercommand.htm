sendGameServerCommand( <clientnum>, <command> )

Module: Utils

Summary:

Sends a game server command to the client, where it is parsed in CG_ServerCommand(). Example commands and their use:

"B": calls CG_MapRestart(0)
"C": (int) switchToOffHand()
"D": (int, str, float, float, float) deactivateReverb()
"E": (int, int, float) setChannelVolumes()
"F": (int, str, float, float, float) deactivateChannelVolumes()
"G": (int) setTeamScore() for axis
"H": (int) setTeamScore() for allies
"I": (str) giveWeapon() (str indicates type, includes item pickup stuff)
     (str) setNormalHealth()
"J": (int) calls CL_ResetPlayerMuting() for each disconnecting player
"K": closeIngameMenu()
"a": (int) switchToWeapon()
"b": (int, int, int, ...) scoreboard info (/+score)
        b        
        Example with 2 players:
        b 2 0 0 1 3 29 0 0
                0 0 43 4 0
        Example with 3 Players:
        b 3 0 0 1 0 32 0 0
                2 0 61 0 0 
                0 0 48 1 0
"c": (str) announcement()
"d": (int, str) update of configstrings, e.g., when changing sv_voice, or at the use of functions like:
     (int, str) playSound(), ambientPlay()
     (int, str) playFx(), playFxOnTag()
     (int, int, int, float, float, float, float, int) setExpFog()
"e": (str) error message, like iprintLn(), sent to active players on kick() etc.
"f": (str) iprintLn()
"g": (str) iprintLnBold()
"h": (str) chat message
"i": (str) team chat message
"n": calls CG_MapRestart(1)
"o": (str) musicPlay()
"p": (int) musicStop()
"q": (int) soundFade()
"r": (int, str, float, float, float) setReverb()
"s": (int) playLocalSound()
"t": (int) openMenu()
"u": closeMenu()
"v": (str) setClientCvar()
"w": (str) disconnect message for the player that disconnected


Example:

sendgameservercommand(player getentitynumber(), "h \"console: " + message + "\"");



Required Args:

  • 1 : <clientnum> The client number of the player.
  • 2 : <command> The game server message.
