file_link( <source>, <dest> )

Module: Utils

Summary:

Creates a new link to an existing file. Returns 0 on success or -1 as error.



Example:

linked = file_link( "/home/server/files/gob_aim.iwd", "/home/server/mod/gob_aim.iwd" );



Required Args:

  • 1 : <source> the old path
  • 2 : <dest> the new path
