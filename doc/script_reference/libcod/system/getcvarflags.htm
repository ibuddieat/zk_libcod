getCvarFlags( <cvar> )

Module: System

Summary:

Gets the flags of a cvar (e.g., 0x80 is set for cheat-protected cvars). Returns an error (undefined) if the cvar does not exist.



Example:

// MakeCvarServerInfo check
if ( getCvarFlags("ui_menutext") & 0x100 ) ...



Required Args:

  • 1 : <cvar> The cvar name
