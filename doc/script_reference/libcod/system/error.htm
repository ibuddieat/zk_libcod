error( <message>, <terminal> )

Module: System

Summary:

The function will throw a script error. If the server loaded the current map in developer mode or the optional terminal parameter is set to true, the server will halt.



Example:

error("unexpected value in function x", false);



Required Args:

  • 1 : <message> Error message


Optional Args

  • 1 : <terminal> Flag stating whether the error is terminal (halt server if true). Default: False
