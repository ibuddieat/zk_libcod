findConfigStringIndex( <string>, <min. index>, <max. index> )

Module: System

Summary:

If found, returns the index of a given configstring that is searched within the given index range. Returns true if the configstring could not be found, or at the first empty configstring within the given index range.



Example:

index = findConfigStringIndex("Text", 1310, 1566);



Required Args:

  • 1 : <string> The string to search
  • 2 : <min. index> Start index of configstrings to compare against
  • 3 : <max. index> End index of configstrings to compare against
