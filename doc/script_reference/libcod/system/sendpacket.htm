sendPacket( <address>, <message> )

Module: Player

Summary:

Sends a UDP datagram to the specified address, containing the provided message (prefixed with 0xFFFFFFFF).



Example:

sendPacket("192.168.1.1:28960", "rcon test map mp_toujane\n");



Required Args:

  • 1 : <address> The remote address
  • 1 : <message> The message to send
