getMicroseconds()

Module: Utils

Summary:

Returns the server's start time (integer) in microseconds. Always returns a positive number. The value overflows (i.e., starts again from zero) every ~35 minutes of server uptime. Due to this high precision and therefore low interval time, this function may fit well for benchmarking/debugging tasks, but not for monitoring long-running procedures. However, keep in mind that this function will introduce some overhead on its own.



Example:

time = getMicroseconds();
for(i = 0; i < 1000; i++)
    bulletTrace((0, 0, 0), (0, 0, 8192), true, undefined);
logPrintConsole("Delta: " + (getMicroseconds() - time) + "us\n");
