setCullFogForPlayer( <near distance>, <far distance>, <red>, <green>, <blue>, <transition time> )

Module: Effects

Summary:

Sets the amount of fog in the distance, for the given player only. The fog will increase linearly. Please note that visibility traces through fog only take the global fog settings (see SetCullFog) into account.



Call this on:

The player



Example:

self setCullFogForPlayer(0, 16500, 0.7, 0.85, 1.0, 0);



Required Args:

  • 1 : <near distance> Distance from the camera that the fog will start
  • 2 : <far distance> Distance from the camera that full occlusion will occur
  • 3 : <red> The red component of the fog as a value between 0.0 and 1.0
  • 4 : <green> The red component of the fog as a value between 0.0 and 1.0
  • 5 : <blue> The red component of the fog as a value between 0.0 and 1.0
  • 6 : <transition time> Transition time in seconds
