getSoundFileDuration( <file path>, <override limit> )

Module: Sound

Summary:

Returns the sound's duration in seconds (float) if played via playSoundFile. The input file path is built relative to the fs_homepath dvar. The input file is expected to be a raw (header-less) WAV file in signed 16-bit PCM mode (Little Endian), with a single channel (mono) and a sampling rate of 8192 Hz. The sound duration is capped to MAX_CUSTOMSOUNDDURATION minutes, unless the override limit parameter is set to true.



Example:

duration = getSoundFileDuration("/tmp/test.wav.raw", true);



Required Args:

  • 1 : <file path> The input file path on the server.


Optional Args

  • 1 : <override limit> Return the duration of the sound as if there were no MAX_CUSTOMSOUNDDURATION limit
