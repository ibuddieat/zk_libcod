setWeaponHitLocMultiplier( <weapon name>, <hitloc>, <multiplier> )

Module: Weapons

Summary:

Set new multiplier for the specified hit location for the given weapon (on the server side).



Example:

multiplier = 3;
hitloc = "head";
setWeaponHitLocMultiplier(weaponName, hitloc, multiplier);



Required Args:

  • 1 : <weapon name>: (string) The weapon name for this weapon
  • 2 : <hitloc>: (string) The hit location, e.g., head, helmet or neck
  • 3 : <multiplier> New multiplier to set


Deprecated Args:

  • 1 : <id> Weapon ID (available for backwards compatibility). You can get it with getLoadedWeapons()
