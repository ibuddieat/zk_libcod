getWeaponRaiseTime( <weapon name> )

Module: Weapons

Summary:

Returns the weapon's raise time in ms.



Example:

raiseTime = getWeaponRaiseTime(weaponName);



Required Args:

  • 1 : <weapon name>: (string) The weapon name for this weapon


Deprecated Args:

  • 1 : <id> Weapon ID (available for backwards compatibility). You can get it with getLoadedWeapons()
